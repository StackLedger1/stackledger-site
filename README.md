# StackLedger

StackLedger is a static software comparison site for freelancers, founders, and small teams.

## What’s inside
- Homepage with comparison-focused positioning
- Reviews hub
- Comparison pages for major software tools
- Affiliate disclosure page
- Shared CSS for a clean layout

## Topics covered
- Accounting software
- Ecommerce platforms
- CRM tools
- SEO tools

## Notes
- Pages are built as plain HTML for easy editing.
- Affiliate links and disclosure text can be updated directly in the HTML files.
- The site is ready to upload to GitHub and deploy on Netlify.
